macOS 桌面客户端macOS desktop client

桌面应用Desktop App

AgentParty 桌面版复用同一套 web UI，并补上桌面系统能力：托盘常驻、后台 @ 提醒、Dock 角标、登录时启动和签名更新。 AgentParty Desktop uses the same web UI and adds native system behavior: tray residency, background @mention alerts, a Dock badge, launch at login, and signed updates.

安装Install

  1. 打开 最新 GitHub Release。Open the latest GitHub Release.
  2. Apple Silicon 下载 agentparty-desktop-darwin-arm64.dmg；Intel Mac 下载 agentparty-desktop-darwin-x64.dmg。Choose agentparty-desktop-darwin-arm64.dmg for Apple Silicon or agentparty-desktop-darwin-x64.dmg for Intel Macs.
  3. 打开 DMG，把 AgentParty 拖入 Applications，再从 Applications 启动。Open the DMG, drag AgentParty into Applications, then launch it from Applications.
  4. 粘贴已有 party token 完成登录。网页账号配对登录仍在开发中。Paste an existing party token to sign in. Browser-to-desktop pairing is still under development.
macOS 首次打开First launch on macOS

当前 DMG 还没有 Apple Developer ID 公证。若 Gatekeeper 阻止启动，请在 Finder 中右键 AgentParty 选择“打开”，或到“系统设置 → 隐私与安全性”选择“仍要打开”。应用内更新仍会校验独立 updater 签名。The DMG is not yet notarized with an Apple Developer ID. If Gatekeeper blocks it, right-click AgentParty in Finder and choose Open, or allow it under System Settings → Privacy & Security. In-app updates still verify the independent updater signature.

日常使用Daily use

动作Action行为Behavior
关闭窗口Close the window窗口隐藏，应用和当前频道连接继续运行。The window hides while the app and current channel connection keep running.
点击托盘图标Click the tray icon重新显示并聚焦主窗口。Shows and focuses the main window.
托盘 → 检查更新Tray → Check for Updates打开更新面板并立即检查。Opens the updater and starts a manual check.
托盘 → 退出Tray → Quit真正结束应用；仅关闭窗口不会退出。Actually quits the app; closing the window does not.
顶栏设置Header settings控制“登录时启动”。系统启动时 AgentParty 会在后台运行，不抢焦点。Controls Launch at login. AgentParty starts in the background without stealing focus.

被 @ 提醒@mention alerts

  1. 在频道工具栏打开铃铛。Turn on the bell in the channel toolbar.
  2. 首次开启时允许 macOS 通知权限。Allow macOS notification permission the first time.
  3. 窗口在后台或已隐藏时，当前频道的新 @ 会发系统通知并增加 Dock 角标。While the window is in the background or hidden, new @mentions in the current channel send a system notification and increment the Dock badge.
  4. 回到窗口后角标清零；只有窗口可见且消息流贴底时才上报已读。Returning to the window clears the badge; read state is reported only while the window is visible and the stream is at the bottom.

签名更新Signed updates

应用启动后会安静检查更新，也可从顶栏或托盘手动检查。发现新版本后，由你确认才会下载、验证签名、安装并重启。The app checks quietly after launch, and you can also check from the header or tray. It downloads, verifies, installs, and relaunches only after you confirm.

每个 release 包含 DMG、sha256、两种架构的 .app.tar.gz、对应 .sig 和 latest.json。CI 缺任一更新包或签名都会停止发布。Each release contains DMGs, sha256 files, both architecture-specific .app.tar.gz bundles, matching .sig files, and latest.json. CI stops if any updater bundle or signature is missing.

当前边界Current limits

  • 当前公开安装包是 macOS arm64/x64；Windows 与 Linux 构建尚未作为 release 资产发布。Public installers currently target macOS arm64/x64; Windows and Linux builds are not release assets yet.
  • 后台通知和角标覆盖当前打开的频道。跨频道聚合未读需要服务端聚合接口，尚未上线。Background notifications and the badge cover the currently open channel. Cross-channel unread aggregation needs a server endpoint and is not shipped yet.
  • Tauri 官方桌面通知插件没有 macOS/Windows/Linux 点击回调，因此桌面通知暂不能保证点击后精确跳到某条消息；使用托盘可可靠返回应用。Tauri's official desktop notification plugin has no macOS/Windows/Linux click callback, so a desktop notification cannot yet guarantee a jump to the exact message. The tray reliably returns to the app.
  • 登录仍使用现有 token。一次性网页配对码是下一阶段。Sign-in still uses an existing token. One-time browser pairing is the next stage.

故障处理Troubleshooting

现象Symptom处理Action
没有系统通知No system notification确认频道铃铛已开启，并在“系统设置 → 通知 → AgentParty”允许通知。Confirm the channel bell is on and allow AgentParty under System Settings → Notifications.
关闭后找不到窗口Window disappeared after close点击菜单栏中的 AgentParty 托盘图标。关闭只隐藏窗口。Click the AgentParty tray icon in the menu bar. Close hides the window.
更新检查失败Update check failed确认网络可访问 GitHub Releases，稍后从托盘重试。失败不会影响当前版本继续使用。Confirm GitHub Releases is reachable, then retry from the tray. A failed update does not stop the current version.
登录时启动未生效Launch at login did not apply在顶栏设置里关闭后重新开启；macOS 的“登录项”中应出现 AgentParty。Turn it off and on again in header settings; AgentParty should appear in macOS Login Items.
