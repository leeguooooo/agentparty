团队面板博客风

Issue #504 · 参考 团队面板 博客风.dc.html

设计总纲

结构用 01 分工 / 02 Agent 看板 / 03 协调 三个页签替代长滚动，页签显示未认领和 @ 角标。
视觉延续终端/博客风、等宽字体、单一强调色、方角细边框；长 owner ID 截断，悬停显示全值。
分工已认领成员使用左侧语义色条卡片；编辑控件默认隐藏；未认领成员放进虚线折叠区并可一键带入认领表单。
Agent 看板忙/阻塞为空时使用紧凑虚线占位；空闲卡使用三列网格；离线列表折叠，带交接说明者优先。
协调离开摘要、主机接管和成员筛选按处理顺序排列；命令使用高对比终端块。
实现保持 DivisionBoard、AgentBoardPanel 和协调数据源不变，仅重塑信息架构、默认展开状态和视觉表达。
